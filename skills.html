Продуктовая стратегия

Полная стратегия digital-продукта — от рынка до роадмапа, в интерактивный HTML.

Проводит через весь цикл: рынок → конкуренты → персоны/JTBD → CJM → вижн → роадмап → метрики. Каждая цифра с источником и датой.
  • 10 разделов: рынок (TAM/CAGR), JTBD-персоны, 3 эшелона конкурентов, продукт as-is, вижн, фокусы, роадмап 12 мес, метрики
  • AI-касдев: синтетические интервью — Клод играет персону, кластеризует боль, проверяет гипотезы
  • Два вижина: нишевый/premium vs суперапп/platform + сравнение
  • На выходе: самодостаточный интерактивный HTML с сайдбаром (на русском)
Пример запроса«Построй продуктовую стратегию для нашего LMS-приложения на рынке РФ, горизонт 12 месяцев. Вот известные конкуренты…»

Продуктовый маркетинг

Единый контекст-документ о продукте, аудитории и позиционировании.

Один раз фиксируешь фундамент — и все остальные маркетинг-скиллы автоматически подтягивают этот файл. Не повторяешь вводные в каждой задаче.
  • 12 секций: продукт, ICP, персоны, боли, конкуренты, дифференциация, возражения, голос бренда, proof points, цели
  • JTBD Four Forces: Push / Pull / Habit / Anxiety — динамика переключения
  • Customer Language: дословные цитаты клиентов вместо своих формулировок
  • На выходе: файл .agents/product-marketing.md — общий контекст
Пример запроса«Настрой контекст продукта: опиши наш ICP, позиционирование и боли — сделай черновик из репозитория, потом поправлю.»

Стратегия голубого океана

Уводит из ценовых войн в неоспариваемое рыночное пространство.

Помогает найти новый спрос, где конкуренты нерелевантны — одновременно снижая издержки и повышая ценность.
  • ERRC-сетка: Устранить / Снизить / Повысить / Создать факторы конкуренции
  • Strategy Canvas: кривые ценности — ты vs конкуренты
  • Six Paths + три яруса не-клиентов; оценка стратегии по шкале 0–10
Пример запроса«У нас SaaS для дизайнеров, все режут цены. Найди голубой океан — прогони через ERRC и strategy canvas.»

Брейншторминг идей в дизайн

Превращает сырую идею в согласованную спеку — до кода.

Заставляет прояснить намерение, ограничения и критерии успеха раньше реализации — там, где непроверенные допущения дороже всего.
  • Вопросы по одному → фокус на цель / ограничения / критерии успеха
  • 2–3 подхода с трейд-оффами и рекомендацией; YAGNI безжалостно
  • Гейт: не пишет код, пока дизайн не одобрен → затем спека в файл
Пример запроса«Хочу добавить push-реактивацию для спящих юзеров. Давай сначала разберём, что именно строим.»

Маркетинговые идеи

Подбирает тактики роста из библиотеки 139 проверенных идей.

Застрял, как продвигать продукт? Даёт 3–5 идей под твой этап, бюджет и команду — с первыми шагами и ожидаемым результатом.
  • 139 идей в 17 категориях: контент/SEO, платный трафик, комьюнити, email, партнёрства, PR, PLG…
  • Подбор по этапу (pre-launch → scale), бюджету (0 → high) и таймлайну
  • Подборки под задачу: «нужны лиды срочно», «строим авторитет», «PLG», «enterprise»
Пример запроса«У нас B2B SaaS на early stage, бюджет почти ноль, команда 3 человека. Дай маркетинговые идеи под нас.»

/marketing-psychology

Маркетинговая психология

Применяет когнитивные искажения к маркетинговым решениям.

Объясняет, почему люди покупают, и даёт конкретные рычаги под задачу — низкая конверсия, ценовые возражения, отток, паралич выбора. С оговоркой об этике.
  • Психология покупателя: эффект владения, IKEA-эффект, нулевая цена, парадокс выбора, Зейгарник
  • Убеждение: якорение, эффект приманки (decoy), дефицит, взаимность, фрейминг
  • Цена: charm-pricing, левая цифра, good-better-best; модели поведения: Fogg B=MAP, AIDA
Пример запроса«Юзеры бросают оплату на pricing-странице. Какие психологические модели применить, чтобы поднять конверсию?»

03 · Установка

Как подключить

Честно: набор из трёх источников, единой команды нет. Скилл — это папка с файлом SKILL.md. Плагины ставятся через /plugin, личные скиллы просто лежат в ~/.claude/skills/.

superpowers

brainstorming

Плагин superpowers (официальный marketplace Anthropic). Ставится один раз, внутри — десятки скиллов.

/plugin marketplace add anthropics/claude-plugins-official /plugin install superpowers@claude-plugins-official
wondelai-skills

blue-ocean-strategy

Из marketplace wondelai (набор strategy-growth — вместе с Crossing the Chasm, Traction, Obviously Awesome).

/plugin marketplace add wondelai/skills /plugin install strategy-growth@wondelai-skills
~/.claude/skills

ещё 4 скилла

product-strategy, product-marketing, marketing-ideas, marketing-psychology — личные скиллы. Просто папка + SKILL.md.

~/.claude/skills/product-strategy/SKILL.md ~/.claude/skills/marketing-ideas/SKILL.md # положи файл — скилл готов
Проверить, что подключилось: набери / в Claude Code — скилл появится в списке. Вызывается командой /имя-скилла, дальше Клод сам ведёт по шагам.
04 · Практика

Как применять — два маршрута

Скиллы складываются в цепочку. Сначала фундамент, потом стратегия и упаковка.

🧭 Маршрут продакта

  • 1/product-marketing — зафиксируй контекст продукта, ICP и боли. Один раз.
  • 2/product-strategy — собери стратегию с рынком, персонами и роадмапом в интерактивный HTML.
  • 3/blue-ocean-strategy — проверь позиционирование: где уйти от конкуренции через ERRC.
  • 4/brainstorming — любую новую фичу прогоняй здесь до реализации.

📣 Маршрут маркетолога

  • 1/product-marketing — тот же контекст-файл: его подтянут все маркетинг-скиллы.
  • 2/marketing-ideas — выбери каналы роста под этап и бюджет.
  • 3/marketing-psychology — усиль конверсию: якорение, decoy, фрейминг цены.
  • 4Дальше — в профильные скиллы: ads, copywriting, cro, emails.
